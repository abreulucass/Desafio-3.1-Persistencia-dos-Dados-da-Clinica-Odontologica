(static) cadastrarPaciente(consultorio) → {boolean}

Realiza o cadastro de um novo paciente. Verifica se o CPF informado é válido, se o paciente já existe no consultório, se o nome é válido, e se a data de nascimento está correta. Também calcula a idade do paciente.
Parameters:
Name Type Description
consultorio Object Objeto representando o consultório onde o paciente será cadastrado.
