(static) cadastrarConsulta(consultorio) → {boolean}

Realiza o cadastro de uma nova consulta no sistema. Verifica se o paciente existe, se não há agendamento anterior, se os dados fornecidos são válidos (data, hora), e se o agendamento não está sobrepondo outros já existentes.
